# SubZero Gym MLX Models

On-device AI models for SubZero Gym iOS app, optimized for Apple Silicon using MLX.

## Models

| Model | Size | Target Devices | Download |
|-------|------|----------------|----------|
| Qwen 2.5 3B Instruct (4-bit) | ~1.8GB | iPhone 15 Pro+, iPhone 16 series, iPhone 17 series (8GB+ RAM) | [v1.0.0](https://github.com/ShadowPing-LLC/subzero-gym-models/releases/download/v1.0.0/qwen2.5-3b-instruct-4bit.zip) |
| Qwen 2.5 1.5B Instruct (4-bit) | ~900MB | iPhone 14 series, iPhone 15/15 Plus (6GB RAM) | [v1.0.0](https://github.com/ShadowPing-LLC/subzero-gym-models/releases/download/v1.0.0/qwen2.5-1.5b-instruct-4bit.zip) |

## Purpose

These models power on-device AI features in SubZero Gym:
- Pre-generation profile validation
- Equipment substitution suggestions
- Exercise swap recommendations
